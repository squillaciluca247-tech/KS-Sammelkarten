# Motorrad Club Card Generator — Canvas V1

Diese Version rendert die Karte direkt in ein Canvas.

## Vorteil

- Vorschau = PNG/JPEG/PDF
- Kein html2canvas
- Kein html-to-image
- Keine schwarzen Exporte
- Keine verzerrten Bilder
- Bildposition über X/Y/Zoom steuerbar

## Start

```bash
npm install
npm run dev
```

## Vercel

- Application Preset: Next.js
- Install Command: npm install
- Build Command: npm run build
- Output Directory: leer lassen
- Root Directory: leer lassen, wenn package.json direkt sichtbar ist
